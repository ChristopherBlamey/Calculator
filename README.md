# Calculadora de Ingredientes – Carrito Chileno

Esta es una aplicación web profesional diseñada para ayudar a los vendedores de comida callejera en Chile a calcular las cantidades de ingredientes necesarias para sus productos.

## Características

- **Cálculo de Ingredientes:** Selecciona entre diferentes tipos de productos (Completos, Hamburguesas, Churrascos, As).
- **Personalización:** Ajusta las porciones de ingredientes según sea necesario.
- **Lista de Compras:** Genera una lista de compras detallada basada en las cantidades calculadas.
- **Estimación de Costos:** (En desarrollo) Ayuda a estimar el costo total de los ingredientes.

## Tecnologías Utilizadas

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Componentes:** shadcn/ui
- **Estado:** Zustand

## Instalación y Desarrollo

Primero, clona el repositorio e instala las dependencias:

```bash
npm install
```

Luego, inicia el servidor de desarrollo:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
